# ⚠️ ВАЖНО: Первый запуск

## Перед запуском Docker:

### 1. Активируйте бота в Telegram

**Найдите вашего бота** и **нажмите Start (Начать)**

Без этого шага бот не сможет отправлять вам сообщения!

Ваш бот: `@YourBotUsername`

### 2. Проверьте настройки

Убедитесь что в `.env` правильно указаны:
- ✅ `TELEGRAM_BOT_TOKEN` - токен от @BotFather
- ✅ `TELEGRAM_CHAT_ID` - ваш ID от @userinfobot
- ✅ `WIZZAIR_URL` - ссылка на нужный рейс
- ✅ `FLIGHT_DATE` - дата вылета

## Запуск:

```bash
# 1. Запустите Docker Desktop
open -a Docker

# 2. Дождитесь пока Docker запустится

# 3. Запустите проект
docker-compose up --build
```

## Первый скриншот:

Придёт через **15-20 секунд** после запуска!

## Если скриншот не пришёл:

### Проверьте логи:
```bash
docker-compose logs -f
```

### Типичные ошибки:

**"chat not found"** → Вы не начали диалог с ботом  
**"Unauthorized"** → Неправильный токен бота  
**Connection error** → Проблемы с интернетом

## Остановка:

```bash
docker-compose down
```

## Тестирование без Docker:

```bash
# Активируйте виртуальное окружение
source .venv/bin/activate

# Запустите тест
python test_screenshot.py
```
